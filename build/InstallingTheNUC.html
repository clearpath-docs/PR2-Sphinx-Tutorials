Installing an Image to the NUC¶

The process to install a previously-cloned image to the NUC is quite simple. All that is required are two USBs.

  1. Create a bootable Clonezilla Live USB (http://clonezilla.org/clonezilla-live.php#make).
  2. Download the pr2-head tarball from the PR2 packages server (http://www.clearpathrobotics.com/pr2-packages/pr2-releases) and unpack it on another USB.
  3. Plug in the USB into the NUC and boot off of the Clonezilla Live USB. Press F10 when the NUC boots up to enter into the boot media menu and select the Clonezilla Live USB.
  4. Select the Clonezilla Live setting in the menu and wait for the next menu to show up.
  1. Select a language to use (use arrow keys and enter).
  2. Select “Don’t touch keymap” on the “Configuring console-data” menu.
  3. Select “Start_Clonezilla”.
  1. Select “device-image” to clone the image onto your NUC (Note the hint: press space to mark selections on menu with multiple options)
  1. Choose “local_dev” from the “Mount Clonezilla image directory” menu.
  1. Plug in the non-Clonezilla USB, wait 5 seconds, then press enter.
  1. Select the USB from the list of disks. It will be the volume listed as something like “sdb1” or “sdc1”.
  1. Select the “/” directory from next menu.
  2. After a list of information about the disk, press enter then select the “Beginner” option.
  3. Select the “restoredisk” option from the next menu. Ensure you select the correct option on this step!
  1. Select the pr2-head image file and the drive you want to clone the image to, then press enter to continue.
  2. Press “y” then enter for the next few prompts.
  3. Wait until the process completes by itself.
  4. After the process completes, the following screen will show up. Press enter.
  1. Select “poweroff” and turn off the NUC. After the NUC is off, remove the two USBs.
  1. Power on the NUC. You’re done! Your NUC should now be fully functional!
